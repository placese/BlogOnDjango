# BlogOnDjango
## To run this web app:
1) ```virtualenv venv```
2) ```source venv/bin/activate```
3) ```pip install -r requirements.txt```
4) ```python manage.py runserver```
### To turn on DEBUG mode:
```blogengine/blogengine/settings.py/ -> DEBUG = True```
### To add a super user:
```python manage.py createsuperuser```
